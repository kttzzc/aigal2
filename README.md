# AIgal — 由 AI 驱动的视觉小说游戏

> [English Version](README_EN.md)

> 🎮 欢迎来到 AIgal！这是一个由大语言模型（LLM）实时驱动的 Galgame（视觉小说）游戏。
> 传统的 Galgame 虽然剧情精彩，但往往只有固定的几条选项分支，玩家只能做被动的"选择题"，一旦通关便很难再有新鲜感。
> 而在 AIgal 中，**没有既定的剧本，也没有死板的选项**！你可以用自然语言输入任何你想说的话、想做的事，AI 会根据你的举动实时生成后续的剧情发展、人物对话，并自动为你切换场景背景和角色立绘。你的每一次选择、每一次游玩，都将展开一段独一无二、拥有无限分支的沉浸式冒险。

## 🚀 快速启动

### 环境要求

| 依赖 | 版本 |
|------|------|
| Node.js | ≥ 18 |
| Python | ≥ 3.10 |
| npm | ≥ 9 |

### 一键启动

双击根目录下的 `START.bat` 即可同时启动前后端：

```bat
START.bat
```

启动后：
- 前端界面：[http://localhost:5173](http://localhost:5173)
- 后端 API：[http://localhost:8000](http://localhost:8000)
- API 文档：[http://localhost:8000/docs](http://localhost:8000/docs)

### 手动启动

**后端**

```bash
cd backend
pip install -r requirements.txt
uvicorn src.main:app --reload
```

**前端**

```bash
cd frontend
npm install
npm run dev
```

### 首次使用配置

1. 启动后进入游戏，打开 **设置**（右上角菜单 → 设置，或实验性 UI 下点击 System）
2. 在 **AI 配置** 中填写你获取到的 API Key、Base URL 以及对应的模型名称。

## 📖 详细介绍

### 核心游戏循环

```
玩家输入想法/对话 → AI 实时生成 JSON 响应 → 游戏解析并渲染对话、角色、背景 → 等待玩家下一步行动
```

AI 会以结构化的 JSON 格式返回内容，游戏系统负责将其渲染为视觉小说的画面。每次 AI 响应可能会包含**多段对话**，玩家可以像玩传统 Galgame 一样逐条点击推进阅读。

### AI 返回的 JSON 结构参考

```json
{
  "messages": [
    {
      "id": 1,
      "name": "角色名",
      "message": "对话内容",
      "background": "背景图文件名.png",
      "roles": [
        { "id": 1, "img": "角色立绘.png", "loc": "left" },
        { "id": 2, "img": "角色立绘.png", "loc": "right" }
      ]
    }
  ],
  "variables": {
    "好感度": "50",
    "物品": "魔法书"
  },
  "time": {
    "time_id": 1,
    "con": "本轮剧情内容概述"
  }
}
```

### 核心系统一览

#### 🎭 游戏界面
- **对话框**：精致的玻璃拟态风格，支持逐字打字效果，点击平滑推进
- **角色立绘**：支持多角色同屏互动（左/右/居中）。
- **场景背景**：跟随 AI 的剧情安排，自动无缝切换场景图片
- **底部操作栏**（实验性 UI 下开启）：Save / Load / Time / System / Title 等核心功能常驻底部，方便随时调用

#### 📖 世界书（World Book）
- 允许你为游戏注入专属的世界观和设定！通过关键词匹配，自动在后台将相关设定发送给 AI。
- **常驻触发**：支持设置无需关键词也能始终生效的核心条目。
- 支持将你的世界书设定导出/导入为 JSON 文件，与朋友分享。

#### 📝 剧本导读（Prompt Editor）
- 游戏的系统提示词以 `.md` 文件保存，支持创建多套提示词随时切换。
- 提供默认的 AIgal 角色扮演规则，你可以自由编辑，改变 AI 的叙事风格或游戏玩法。

#### 🎨 素材工坊（Asset Manager）
- 轻松上传和管理你的背景图与角色立绘库。
- 提供可视化的素材预览界面，让 AI 有更多画面资源可供调用。

#### 📊 全局变量记忆（Variables）
- AI 可以自己决定修改游戏内的变量（如：好感度+5、获得物品、解锁成就等），并且这些状态会永久保存。
- 每次对话都会将当前状态回传给 AI，实现拥有真正长期记忆的跨回合交互。

#### ⏱️ 命运时间线（Timeline）
- 随着剧情发展，AI 会自动记录一个个带有递增 `time_id` 的剧情节点摘要。
- 你可以直观地查看自己走过的剧情脉络。
- **时间线回溯（悔棋功能）**：选错对话导致坏结局了？别怕！点击历史节点的回溯按钮，整个世界（包括所有变量和对话历史）都会瞬间倒流回那个时刻！

#### 💾 存档系统（Save / Load）
- 随时保存当前的世界线进度（完整保存对话历史与变量快照）。
- 提供多存档位管理，安全守护你的每一段故事。

#### 🔊 语音合成（TTS）
- 接入 MiniMax 强大的语音大模型接口，支持给 AI 角色配音！
- 可在设置中开启 **自动朗读**，或在对话框右下角随时点击播放当前句子的语音。
- 具备完善的**缓存机制**，听过的语音不会重复消耗 Token。

#### 🐛 开发者模式（Debug Panel）
- 游戏内按 `F12` 唤起专属的内置调试控制台（替代了浏览器原生的开发者工具）。
- 透明化查看每轮对话发往外部大模型的 **原始请求体** 和 **响应体**。
- 支持**热更新**：直接手动修改当前对话的 JSON 数据，点击保存，游戏画面立刻同步你的修改！

#### ⚙️ 丰富的设置项
- AI 节点配置（API Key / Base URL / 模型）
- 自由调整上下文记忆上限（设置为 0 即可将全部历史喂给 AI）
- 可调节文字显示速度、自动播放间隔
- UI 动效开关与布局模式切换

### 两种 UI 交互模式

| 经典模式 | 实验性 UI |
|----------|-----------|
| 右上角汉堡菜单收纳所有功能 | 底部常驻核心操作栏（Save / Load / Time / System / Title） |
| 功能面板以侧边模态弹窗形式打开 | 高级设置（如提示词、世界书等）整合到了独立的 `/settings` 页面中 |
| 默认启用 | 需要在设置中手动开启，更适合重度文字游戏玩家 |

## 🛠️ 技术栈揭秘

### 前端

| 技术 | 用途 |
|------|------|
| **React 18** | 核心 UI 框架 |
| **TypeScript** | 保证大型项目的类型安全 |
| **Vite** | 极速的构建工具与开发服务器 |
| **Zustand** | 轻量且强大的状态管理 |
| **Framer Motion** | 丝滑的 UI 动画与过渡效果 |
| **Axios** | 处理与后端的 HTTP 通信 |
| **React Router** | 优雅的单页路由跳转 |

### 后端

| 技术 | 用途 |
|------|------|
| **Python 3.10+** | 核心运行时 |
| **FastAPI** | 高性能的 Web API 框架 |
| **Uvicorn** | 闪电般的 ASGI 服务器 |
| **httpx** | 异步请求大模型 API，拒绝阻塞 |
| **Pydantic** | 严谨的数据校验与模型序列化 |

### 纯本地的数据存储

- 采用纯本地的 JSON 文件存储方案，**无需配置任何繁杂的数据库**！
- 你的世界书、游戏存档、系统设置等私密数据，均安静地躺在 `backend/data/` 目录下，彻底保障数据隐私。
